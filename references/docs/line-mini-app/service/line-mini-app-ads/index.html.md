# Place ads in LINE MINI Apps

LINE MINI Apps can be monetized by displaying ads. This page explains the types of ads that can be displayed in LINE MINI Apps and how to display them. Ads can be placed in both verified and unverified MINI Apps, but the service must be provided in Japan.

<!-- table of contents -->

## Ads that can be placed in LINE MINI Apps 

With LINE MINI Apps, you can monetize your apps by displaying [LY Ads Network Display Ads (Web)](https://www.lycbiz.com/jp/partner/adnetwork/ly-ads/) (only available in Japanese). To ensure a safe and secure user experience, LINE MINI Apps only display ads that have been reviewed and approved by LY Corporation, and LY Ads has been designated as the ad network for this purpose.

## How to place LY Ads in LINE MINI Apps 

To place LY Ads Network Display Ads (Web) in LINE MINI Apps, follow the steps below.

If you became a network partner of the former Yahoo! JAPAN Ads by March 31, 2026, or a network partner of the LY Ads Network Display Ads (Web) on or after April 1, 2026, proceed to step 3.

- [1. Check the documentation](https://developers.line.biz/en/docs/line-mini-app/service/line-mini-app-ads/#how-to-place-line-mini-app-ads-step-1)
- [2. Apply to become a network partner](https://developers.line.biz/en/docs/line-mini-app/service/line-mini-app-ads/#how-to-place-line-mini-app-ads-step-2)
- [3. Apply for LY Ads site review](https://developers.line.biz/en/docs/line-mini-app/service/line-mini-app-ads/#how-to-place-line-mini-app-ads-step-3)

We'll explain each step.

### 1. Check the documentation 

First, check the documentation regarding advertising in LINE MINI Apps via the link below. The material includes information on the process of placing advertisements in LINE MINI Apps, the LY Ads site review, and important notes. Once you've checked the material, proceed to step 2 if there're no issues.

[Documentation (PDF)](https://s.yimg.jp/images/listing/partnerportal/document/yahooads_networkpartner_lineminiapp.pdf) (only available in Japanese)

### 2. Apply to become a network partner 

To place ads in LINE MINI Apps, you must become a network partner of the LY Ads Network Display Ads (Web). Apply to become a network partner via the link below. When applying, check the [Partner Network Terms of Use](https://s.yimg.jp/images/marketing/portal/paper/partner_network_terms_of_use.pdf) (only available in Japanese).

[Application form](https://form-business.yahoo.co.jp/claris/enqueteForm?inquiry_type=publisher_pn_application) (only available in Japanese)

<!-- note start -->

**Even if you have used other plans, you must apply for the Network Partner plan**

Even if you have previously used LY Ads Network Display Ads (App) or LY Ads Network Search Ads, you must still submit a separate application to become a network partner of the LY Ads Network Display Ads (Web).

<!-- note end -->

### 3. Apply for LY Ads site review 

Once you've registered as a network partner, you'll be able to log in to the management tool for ads (Network Partner Tool). From this tool, submit the LINE MINI App on which you plan to place ads for the LY Ads site review.

Once you pass the LY Ads site review, you'll be able to issue tags for ad placement in the tool. After that, follow the instructions in the documentation provided by us after approval.
